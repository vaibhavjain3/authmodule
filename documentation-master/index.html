About

AuthModule is a pluggable authentication module for Web applications. It provides the developer with everything that is required to make the web application secure by providing user authentiation. It is developed keeping many authentication modes in mind i.e. in-memory user authentication, Oauth authorisation and user authentication using data from a database. It is created to save your time so that you can take care of the other parts of the application.

A pluggable authentication module is useful for developers for several reasons:

  • It provides a common authentication scheme that can be used with a wide variety of applications.
  • It provides significant flexibility and control over authentication for the application developer.
  • It provides a single, fully-documented module which allows developers to write programs without having to create their own authentication schemes.
