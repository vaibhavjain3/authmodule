Installation

Prerequisites

To use the AuthModule on your local system, you need the following:

  • Apache Maven : It is a Java tool, so you must have Java installed in order to proceed
  • Apache Tomcat Server (Version 8.5.x and higher supported)
  • IDE : All popular development environments for the Java platform support Apache Maven
    • Apache NetBeans IDE
    • Eclipse IDE - M2Eclipse
    • JetBrains IntelliJ IDEA
  • Browsers Supported : Mozilla, Firefox, Google Chrome and Edge
  • Database : You will need a database in your system, in the module we have used MySql

Run the Application

The following steps will need to be taken to directly run the application after cloning it from the Github-Repository :

  • Generate Google credentials for Oauth and add them to application.properties.
  • In a MySQL database (named ‘authmodule’), create a table named ‘user’ using the following command:
    CREATE TABLE ‘authmodule’.‘user’ ( ‘id’ INT NOT NULL,
    ‘user_name’ VARCHAR(45) NOT NULL, ‘password’ VARCHAR(45) NOT NULL,
    ‘roles’ VARCHAR(15) NOT NULL, PRIMARY KEY (‘id’));


  • After populating the table, adjust the database url, username and password in both application.properties and hibernate.cfg.xml.

    • Your Application is ready to run!
