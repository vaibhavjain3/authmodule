Resource Customization

We can customise the resources by making suitable changes in the templates folder. We have considered 2 roles: User and Admin for demonstration purposes and used Thymeleaf to create a template for each page. Instead of a single page for user and admin each, we can have multiple pages associated with each role. So, all the customised pages added under user will have USER access and those under admin will have ADMIN access. We can even customise login, logout, index and error pages if required.
